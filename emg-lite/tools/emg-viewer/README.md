# EMG Viewer & Editor

Vite + TypeScript implementation of EMG-lite Viewer.

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```

## Features
- **Editor**: Controls for `emotion`, `activity`, `speaking`, `intensity`.
- **Viewer**: Displays the avatar based on the current state.
- **Assets**: Uses `public/assets` for PNG files.
